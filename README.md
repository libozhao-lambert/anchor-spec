# ANCHOR v0 (Draft Frozen)

ANCHOR is a non-deterministic presence and continuity signal for AI agents,
designed to be considered under uncertainty in multi-agent systems.

This repository is the canonical location for discussion, clarifications,
and issue tracking for the ANCHOR documentation set.

It does **not** provide a hosted service, registry, or official SDK.

## Canonical Documents

Authoritative ANCHOR v0 documents are published on the website:

- Definition: https://anchornetwork.ai/definition
- Specification: https://anchornetwork.ai/spec
- Whitepaper: https://anchornetwork.ai/whitepaper
- Examples: https://anchornetwork.ai/examples
- Developer Docs: https://anchornetwork.ai/docs

## Status

ANCHOR v0 is **Draft Frozen**.
The semantics and scope of v0 will not change (except for minor typos/formatting corrections).

## How to use this repo

- **Issues**: clarification questions, typos, and concrete doc-level improvements
- **Discussions**: conceptual discussion, integration experiences, and future exploration (v1+; not commitments)

When opening an issue, please reference **ANCHOR v0** explicitly and cite the relevant document section.

## Non-goals

ANCHOR is not:
- a capability claim or performance guarantee
- a routing instruction
- an identity or reputation system
